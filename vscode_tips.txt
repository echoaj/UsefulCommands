------------------- VSCODE LANGUAGE DEVELOPMENT ENVIRONMENT -------------------

Pre-actions
Install Code Runner vscode extension
Enable Code-runner:Ignore Selection in settings

C
Create C file
Download gcc compiler (sudo apt install gcc).  gcc will apear in "usr/bin/gcc" & "bin/gcc"
Installs meta packages, g++/GNU compiler, gcc, dpkg-dev, G++ (sudo apt install build-essential) 
Note:
    Installing C/C++ vscode extension did not allow my code to run
    But I installed it anyways.
    Needed up update my compiler path for c/c++ extension in settings to "usr/bin/gcc"

C#
Install C# vscode extension
TYPE IN TERMINAL: 
    dotnet new console
    dotnet new gitignore

------------------------------- VSCODE SETTINGS -------------------------------

Reformat Code
SHIFT+ALT+F

Disable Folders/Like/This
Preferences -> Settings -> Explorer:Compact Folders - Disable

Create Snippet Rule
Preferences -> Settings -> User Snippets

Turn off autocompletion after .
In settings uncheck Editor: Accept Suggestion On Commit Character

Turn off inline (n) refernces.
Turn off Editor:Code Lense in settings

Turn off selection code runner (tempCodeRunnerFile...)
Turn on Code-runner:Ignore Selection in settings

Change program output show in terminal instead of output
Turn on Run In Terminal in settings located within Run Code configuration
