------------------------------- VSCODE SETTINGS -------------------------------

Reformat Code
SHIFT+ALT+F

Disable Folders/Like/This
Preferences -> Settings -> Explorer:Compact Folders - Disable

Create Snippet Rule
Preferences -> Settings -> User Snippets

Turn off autocompletion after .
In settings uncheck Editor: Accept Suggestion On Commit Character

Turn off inline (n) refernces.
Turn off Editor:Code Lense in settings

Turn off selection code runner (tempCodeRunnerFile...)
Turn on Code-runner:Ignore Selection in settings
